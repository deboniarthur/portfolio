Acompanhamento de Perfomance Interna do Time de Suprimentos

Descrição do Projeto

Desenvolvimento de um conjunto de dashboards para analisar indicadores e acompanhar perfomance do time de suprimentos, utilizando Python (APIs), VBA, Excel, Sheets e Looker Studio.

Detalhes Técnicos

Este projeto envolveu a criação de um pipeline completo de dados desde a extração das informações de várias APIs internas até a visualização em dashboards interativos. Utilizei Python para criar scripts que faziam requisições periódicas, transformavam os dados e os armazenavam em um banco de dados SQL.

Para automatizar processos adicionais, implementei macros em VBA que eram acionadas em horários específicos para gerar relatórios complementares. Os dashboards finais foram criados no Looker Studio, conectados diretamente aos bancos de dados e atualizados em tempo real.

Resultados

  • Redução de 30% no tempo de geração de relatórios semanais
  • Identificação de gargalos no processo de suprimentos, resultando em economia de R$250.000/ano
  • Melhoria na precisão dos dados de 78% para 98%

Power BI DAX SQL

Acompanhamento de Orçado da VP

Descrição do Projeto

Criação de um dashboard em Power BI para monitoramento em tempo real das métricas de vendas de uma empresa de e-commerce.

Detalhes Técnicos

O projeto teve como objetivo centralizar todas as métricas financeiras da vice-presidência em um único painel interativo. Desenvolvi um modelo de dados em Power BI que se conectava diretamente ao banco de dados SQL Server da empresa, atualizando os dados a cada hora.

Utilizei DAX (Data Analysis Expressions) para criar medidas complexas que calculavam variações em relação ao período anterior, cumprimento de metas e projeções baseadas em tendências históricas. Implementei também uma série de filtros dinâmicos que permitiam visualizar os dados por período, região, departamento ou produto.

Resultados

  • Centralização de 15 relatórios distintos em um único painel
  • Redução de 8 horas semanais no tempo de análise da equipe financeira
  • Identificação de discrepâncias que resultaram em economia de R$500.000

Sistema de Recomendação de Produtos

Descrição do Projeto

Implementação de um sistema de recomendação de produtos para um site de varejo, utilizando algoritmos de filtragem colaborativa.

Detalhes Técnicos

Desenvolvi um sistema de recomendação baseado em similaridade entre usuários (user-user collaborative filtering) que analisava o histórico de compras e navegação dos clientes. O algoritmo processava os dados usando a biblioteca Scikit-learn, calculando a similaridade cosseno entre perfis de usuários.

Para lidar com o problema de inicialização a frio (cold start), implementei também um sistema híbrido que complementava as recomendações com uma abordagem baseada em popularidade para novos usuários ou produtos. O modelo foi treinado com dados históricos de 2 anos de transações e testado usando validação cruzada.

Resultados

  • Aumento de 5,3% na taxa de conversão de produtos recomendados
  • Redução de 12% na taxa de devoluções
  • Tempo médio de processamento de 0,8 segundos por recomendação
